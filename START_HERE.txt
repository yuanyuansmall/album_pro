🚀 快速启动步骤

第 1 步：配置环境变量
  1. 复制 .env.example 为 .env.local
  2. 填入 Supabase 凭证

第 2 步：设置 Supabase
  1. 访问 https://supabase.com
  2. 创建新项目
  3. 执行 database_init.sql
  4. 创建 'photos' 存储桶
  5. 复制 URL 和 Key

第 3 步：启动开发
  npm start
  访问 http://localhost:3000
